H5

HTML5

简介

H5范称=html+js+css
html5 新增了很多的标签.
css:出现很多新的样式，颠覆性的样式，可以用来做出非常绚丽的界面效果.
js:使用浏览器给我们提供的api，方法，属性，对象。

前端就业方向：

  • pc 端的界面布局，js 处理
  • 移动web 网页
  • 服务端
  • android app ios 的app
  • Canvas 画布（画界面，这个东西可以用来做游戏）方向 flash
    在移动做移动web，js 肯定可以调用手机里面的一些功能.

Web应用开发的一系列标准 (兼容性)

  • 为什么会造成兼容性：
    • w3c 一个组织，指定了web 规范的组织，在前端可以理解为html，css，js 的规范。
    • 每个浏览器都需要有一个内核,浏览器内核又可以分成两部分：
      • 渲染引擎和 JS 引擎。
      • 1.渲染引擎负责取得网页的内容（HTML、XML、图像等等）、整理讯息（例如加入 CSS 等），以及计算网页的显示方式，然后会输出至显示器或打印机。
      • 2.JS 引擎则是解析 Javascript 语言，执行 javascript 语言来实现网页的动态效果。
    • 主流浏览器的内核
      • IE trident
      • 火狐gecko
      • 谷歌苹果webkit
      • Opera：以前是Presto 现在是Blink内核
  • pc 端使用h5
    • ie9包含ie9以上的版本
    • firefox
    • google 没问题
  • 移动端浏览器：基本不用怎么去考虑兼容问题,由于是统一的内核webkit

html5 兼容问题的处理

  • 如果电脑上面没有IE9以下，可以装一个软件，叫做ietest，它可以模拟ie 的各种内核
  • 需要引入一个 html5shiv.min.js 文件，去屏蔽掉兼容性

html新元素:

  • 语义标签(ul li table)
  • 智能表单 (form ) 表单的展现形式更加丰富.
  • 多媒体 (音频，视频)
  • js 增强：
    还提供了一些Javascript API 地理定位，我现在可以在网页里面可以定位你的位置就可以做一些服务 （lbs location base service ）基于位置的服务
    1.附近的人
2.外卖
3.重力感应
4.硬件访问:js 可以硬件.但是也会做一些安全限制.
5.可以在浏览器内实现类原生应用,

语义规范：

  • html4 的文档声明
  • html 5 的文档声明
  • 优点：html5 的声明特别简洁，直观

为什么用语义标签？

  • 就是因为我们的网页需要被第三方的程序阅读，比如屏幕阅读器
  • 我们的网页以后需要推广，一般都是在百度上面推广，百度它会检索我们的网页，然后记录到百度的后台，然后我们搜索，如果我们使用html5 的语义标签，百度的程序（蜘蛛）更方便的去解析，这样如果使用html5 ，我们的网页位置会更加的靠前。

表单

HTML5在Web表单方向也做了很大的改进，如拾色器、日期/时间组件等，使表单处理变的更加高效。

  • 我们跟服务器进行交互有三种方式
    • 表单提交
    • 超链接
    • 直接在地址栏输入 （js 跟服务器提交）
  • 表单提交 (表单是让用户填写)
    • 定义一个表单我们使用form
    • 我们使用 input 来做表单项
    • type 属性用来控制表单的类型

html5的DOM扩展

DOM（document object Model ）文档对象模型

一个html 页面，对应有一个document 对象，操作document对象就是在操作html 的内容.

dom 对象：打开浏览器，浏览器的内核去解析html 网页，解析html 网页的是从上到下解析，解析html 的过程当中，会创建一个dom 对象，把html 里面内容映射到dom 里面来。

在dom 看来，html 里面的内容都是节点,有五种类型
1：文档节点
整个html 可以当作成一个节点
2：元素节点 1
页面里面的所有的标点都可以当作成元素节点
3：属性节点 2
页面上面所有的标签对应的属性都可以理解成属性节点
4：文本节点 3
页面里面所有的标签对应内容都是文本 节点
5：注释节点

做开发都是要操作这些节点，对这些节点进行增删改查，以及样式操作。

我们之前学习了这样的三个方法：

  • document.getElementById 这个返回的是一个对象
  • doccument.getElementsByTagName 根据标签名称去获取，返回的是数组也就是不能直接操作，需要遍历了之后才能操作
  • doccument.getElementsByClassName 返回伪数组，有兼容性 根据class 名称去获取，可以使用索引 [ ]

H5中dom 新增了两个方法

  • document.querySelector 非常常用 根据CSS选择器 返回第一个元素
  • document.querySelectorAll 非常常用 根据CSS选择器 返回所有元素 伪数组

类名classList的增删改查 add remove toggle contains

1、Node.classList.add(‘class’) 添加class

自定义属性

在HTML5中我们可以自定义属性，其格式如下data-*=””，例如:

data-info=”我是自定义属性”，通过Node.dataset[‘info’] 我们便可以获取到自定义的属性值。

Node.dataset是以对象形式存在的，当我们为同一个DOM节点指定了多个自定义属性时，Node.dataset则存储了所有的自定义属性的值。
