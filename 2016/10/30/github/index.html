github

提供共享仓库的远端

操作

  1. 与git的关系: 使用git操作 github提供远端仓库共享服务
  2. github注册
  • 注册后 设置SSH keys 取得权限 创建公私钥
    • 在本地目录 git bush 中输入ssh-keygen -t rsa + 回车
    • 上传公钥 上传至 github网站即可实现免密码登录

  1. push与pull

    • A上传git push gitSHH路径 分支 至github
    • B下载git pull gitSHH路径 分支

  2. 简短的github仓库路径

    • git remote add 名字 gitSHH路径
    • 下次就可以直接使用 例如：
      • git push 名字 分支
    • 显示所有远程主机信息
      • git remote show
  3. clone
    • git clone 仓库地址
    • 相当于创建了一个目录、 进入目录、 执行git init 、git remote add origin 仓库地址
      git pull仓库地址 master
    • clone 后的文件 自定义成origin 可以直接使用 git push origin master

使用

  • start 点赞
  • fork 复制
    别人的项目不允许直接修改 可以先fork一份来到自己的仓库
    修改副本，测试成功后，可以申请原有项目合并副本，经过同意合并操作。
