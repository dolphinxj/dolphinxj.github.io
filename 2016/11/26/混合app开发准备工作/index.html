混合app开发准备工作

jdk安装和配置

下载地址：

环境变量配置

一、java

  • 系统变量增加：JAVA_HOME
    • 把jdk的安装路径复制即可

  • 系统变量增加：CLASSPATH
    • 值为
      .;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar;
      注意开头的一个点

  • 在Path系统变量的后面增加
    • ;%JAVA_HOME%\bin;

  • 命令javac正常运行即可

二、android

  • 下载SDK Tools
    下载地址：

  • 下载完毕，安装或者解压后，在相应目录下找到并打开SDK Manager.exe


SDK Manager默认勾选了一些选项，直接点击Install packages安装即可


如果安装过慢，可以用国内的镜像，或者分包下载
下载地址：


找到相应的包下载即可：
下载地址：

  • Android环境变量配置：
    • 找到SDK Manager所在的目录
      上面下载的包都在同一目录下

+ 新建系统变量：Android

把platform-tools和tools这两个文件夹的路径添加到变量值里面，中间用分号隔开

+ 在Path系统变量的后面增加------  ;%Android%


+ 命令adb正常运行即可

三、安装ionic和cordova

  • 通过npm全局安装

  • 测试是否成功安装

  • 如果npm安装过慢或不成功，可以试试国内淘宝的npm镜像

四、创建项目、打包、发布

  • 创建项目：ionic start appName

创建过程会问是否创建ionic.io账号，n即可

  • 进入创建好的项目目录下，增加android平台

成功增加后会在项目下增加一个platforms的文件夹

  • 打包：在项目下用命令：ionic build android 打包

下载的过程会比较久

网络不好的时候会直接报错误

所以直接拷贝一份成功打过包的.gradle文件夹到本机用户名下就好：

成功打包后，会在platforms/android下增加一个build文件夹

打包好的apk在platforms/android/build/outputs/下，可以拷贝到手机上安装了

  • 直接通过usb数据线在手机上安装

    • 用数据线连上手机，输入adb devices命令，测试手机是否成功连接

+ 运行ionic run android直接安装打过包的apk到手机上


+ 真机演示效果
