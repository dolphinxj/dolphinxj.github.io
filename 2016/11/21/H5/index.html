H5

HTML5

简介

H5范称=html+js+css
html5 新增了很多的标签.
css:出现很多新的样式，颠覆性的样式，可以用来做出非常绚丽的界面效果.
js:使用浏览器给我们提供的api，方法，属性，对象。

前端就业方向：

  • pc 端的界面布局，js 处理
  • 移动web 网页
  • 服务端
  • android app ios 的app
  • Canvas 画布（画界面，这个东西可以用来做游戏）方向 flash
    在移动做移动web，js 肯定可以调用手机里面的一些功能.

Web应用开发的一系列标准 (兼容性)

  • 为什么会造成兼容性：
    • w3c 一个组织，指定了web 规范的组织，在前端可以理解为html，css，js 的规范。
    • 每个浏览器都需要有一个内核,浏览器内核又可以分成两部分：
      • 渲染引擎和 JS 引擎。
      • 1.渲染引擎负责取得网页的内容（HTML、XML、图像等等）、整理讯息（例如加入 CSS 等），以及计算网页的显示方式，然后会输出至显示器或打印机。
      • 2.JS 引擎则是解析 Javascript 语言，执行 javascript 语言来实现网页的动态效果。
    • 主流浏览器的内核
      • IE trident
      • 火狐gecko
      • 谷歌苹果webkit
      • Opera：以前是Presto 现在是Blink内核
  • pc 端使用h5
    • ie9包含ie9以上的版本
    • firefox
    • google 没问题
  • 移动端浏览器：基本不用怎么去考虑兼容问题,由于是统一的内核webkit

html5 兼容问题的处理

  • 如果电脑上面没有IE9以下，可以装一个软件，叫做ietest，它可以模拟ie 的各种内核
  • 需要引入一个 html5shiv.min.js 文件，去屏蔽掉兼容性

html新元素:

  • 语义标签(ul li table)
  • 智能表单 (form ) 表单的展现形式更加丰富.
  • 多媒体 (音频，视频)
  • js 增强：
    还提供了一些Javascript API 地理定位，我现在可以在网页里面可以定位你的位置就可以做一些服务 （lbs location base service ）基于位置的服务
    1.附近的人
    2.外卖
    3.重力感应
    4.硬件访问:js 可以硬件.但是也会做一些安全限制.
    5.可以在浏览器内实现类原生应用,

语义规范：

  • html4 的文档声明
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN"
        "http://www.w3.org/TR/html4/loose.dtd">
        <!--这个里面写html5的一些标签的没有提示-->
<html>
  • html 5 的文档声明
<!DOCTYPE html>
<!--这个里面写html5的一些标签的有提示-->
  • 优点：html5 的声明特别简洁，直观

为什么用语义标签？

  • 就是因为我们的网页需要被第三方的程序阅读，比如屏幕阅读器
  • 我们的网页以后需要推广，一般都是在百度上面推广，百度它会检索我们的网页，然后记录到百度的后台，然后我们搜索，如果我们使用html5 的语义标签，百度的程序（蜘蛛）更方便的去解析，这样如果使用html5 ，我们的网页位置会更加的靠前。

表单

HTML5在Web表单方向也做了很大的改进，如拾色器、日期/时间组件等，使表单处理变的更加高效。

  • 我们跟服务器进行交互有三种方式
    • 表单提交
    • 超链接
    • 直接在地址栏输入 （js 跟服务器提交）
  • 表单提交 (表单是让用户填写)
    • 定义一个表单我们使用form
    • 我们使用 input 来做表单项
    • type 属性用来控制表单的类型

一、 新增表单输入类型：

类型 使用示例 含义 样式
★ email <input type="email"> 输入邮箱格式
★ tel <input type="tel"> 输入手机号码格式
★ url <input type="url"> 输入url格式
★ number <input type="number"> 输入数字格式
★ search <input type="search"> 搜索框（体现语义化）
range <input type="range"> 自由拖动滑块
color <input type="color"> 拾色器
time <input type="time">
date <input type="date">
datetime <input type="datetime">
month <input type="month">
week <input type="week">

二、新增表单元素

元素 含义
★ <datalist> 数据列表
<keygen> 生成加密字符串
<output> 输出结果
<meter> 度量器

三、新增表单属性

属性 用法 含义
★placeholder <input type="text" placeholder="请输入用户名"> 占位符
★autocomplete <input type="text" autocomplete="off"> 自动完成，用于表单元素，也可用于表单自身
autofocus <input type="text" autofocus> 自动获得焦点
multiple <input type="file" multiple> 文件上传多选或多个邮箱地址
form <input type="text" form="某表单ID"> 指定表单项属于哪个form，处理复杂表单时会需要
novalidate <form novalidate></form> 关闭验证
required <input type="text" required> 必填项
pattern <input type="text" pattern="\d"> 自定义验证 正则表达式

例子：
必填手机号:
<input type="tel" name="tel" required="required" pattern="^(\+86)?1[3,5,8](\d{9})$">

datalist可以定义一个option集合 达到一个多选既视感

<input type="text" list="options" placeholder="选择喜欢的水果" />
<datalist id="options">
    <option type="西红柿"></option>    
    <option type="苹果"></option>
    <option type="香蕉"></option>
    <option type="西瓜"></option>
</datalist>

video和audio都有以下属性：
- controls（是否显示默认播放框）
- loop（循环 可以添加数字表示次数）
- autoplay（自动播放 on/off）

DOM（document object Model ）文档对象模型

  • 获取元素
    • document.getElementsByClassName比较常用 根据类名 返回伪数组 可以使用索引 [ ]
    • document.querySelector非常常用 根据CSS选择器 返回第一个元素
    • document.querySelectorAll非常常用 根据CSS选择器 返回所有元素 伪数组
  • 类名操作
    类名classList的增删改查 add remove toggle contains
